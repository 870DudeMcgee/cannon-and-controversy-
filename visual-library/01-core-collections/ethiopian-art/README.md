# Ethiopian Art

## Collect First

- manuscript miniatures
- cross motifs
- border geometry
- evangelist portraits
- saint panels
- red, gold, green, and indigo combinations
- Ge'ez manuscript ornament

## Strong Search Phrases

- ethiopian illuminated manuscript public domain
- ethiopian gospel miniature
- geez manuscript illumination
- ethiopian orthodox icon manuscript
- ethiopian cross pattern manuscript

## Promote To Curated If

- the geometry can be reused as borders or frame elements
- the color relationships feel ceremonial and aged
- the image can be cropped into multiple reusable pieces
- it supports the site's codex and miniature language

## Motif Links

- illuminated borders
- codex-opening transition
- red/gold text reveals
