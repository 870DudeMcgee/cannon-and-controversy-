# Manuscripts

## Collect First

- illuminated folios
- gospel openings
- rubricated text blocks
- gold initials
- vellum macro textures
- page edges and binding details
- uncial and majuscule hands

## Strong Search Phrases

- illuminated gospel manuscript folio
- ethiopian gospel manuscript
- rubricated parchment page
- vellum codex detail
- gold leaf manuscript macro

## Promote To Curated If

- the page has strong texture even under dark grading
- the ornament survives cropping
- the text block can sit behind titles or quotes
- the image can support codex-opening or parchment-reveal motifs

## Motif Links

- glowing parchment reveal
- codex-opening transition
- illuminated borders
- red/gold text reveals
