# Episode Intake Layer

This folder is the bridge between reusable library work and an actual episode build.

Use it only after an asset, generated output, or text variant has survived review.

## Purpose

The visual library stores reusable language.
The episode intake layer turns approved material into episode-specific buckets.

## Fast Workflow

1. Start in `visual-library/ai-review-dashboard.html`.
2. Approve a generated or sourced asset.
3. Duplicate the episode intake templates for the target episode.
4. Place each approved item into one shot bucket only.
5. Record why it exists in the episode package manifest.

## Buckets

Use these buckets consistently:
- `01-opener`
- `02-historical-context`
- `03-quote-witness`
- `04-map-sequence`
- `05-canon-conflict`
- `06-transition-atmosphere`
- `07-chapter-breaks`
- `08-outro`

## Rule

Do not move raw experiments directly into episode folders.
Only approved outputs enter this layer.
