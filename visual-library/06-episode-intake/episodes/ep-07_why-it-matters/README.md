# Episode Folder Template

This is the reusable folder blueprint for every episode package.

Use the naming convention in `episode-folder-convention.md`, then duplicate this structure under `06-episode-intake/episodes/`.

Required root files for each new episode:
- `01_package.md`
- `02_production.md`
- `03_script.md`
- `04_visual_beats.md`
- `05_assets_text.md`
- `06_shot_bucket.csv`

This starter folder is the future-facing standard.

It deliberately consolidates all on-screen text into `05_assets_text.md` so editing does not depend on multiple parallel text files.

Required bucket folders:
- `01-opener/`
- `02-historical-context/`
- `03-quote-witness/`
- `04-map-sequence/`
- `05-canon-conflict/`
- `06-transition-atmosphere/`
- `07-chapter-breaks/`
- `08-outro/`
