# Episode 09 README

Episode 09 closes the documentary block by tracing the canon timeline from early Hebrew stabilization and the Septuagint through Yavneh-era consolidation, patristic witnesses, late antique councils, Jerome, Luther, Trent, Westminster, and the nineteenth-century print cutoff.

Use this folder as the working package for:
- episode framing and approved inputs in `01_package.md`
- production structure and first-cut rules in `02_production.md`
- spoken draft in `03_script.md`
- first assembly order in `04_visual_beats.md`
- on-screen text in `05_assets_text.md`
- shot inventory in `06_shot_bucket.csv`

Narrative role in the series:
- follows Episode 08's concrete inventory of the removed books
- closes the nine-episode core arc with the chronology of canon formation
- lands the final claim that the canon was shaped by a chain of decisions across centuries

- `ep-09_how-the-canon-was-shaped`
