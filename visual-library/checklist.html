Low-Overwhelm Mode

What to do when you come back.

This page is the simplest operating surface in the library. Use it when you do not want to think about structure and just need the next few actions.

Pick one lane, do one small batch, log it, and stop. That is enough.

The only rule

If you are feeling scattered, do not browse the whole library. Choose one lane, finish one small task, and leave the rest alone.

Choose One Lane

I need the overall look

  • Open `style-board.html`
  • Review tone, parchment, codex, overlays
  • Collect only one related category

I need maps or places

  • Open `map-board.html`
  • Use the Jerusalem map workflow
  • Collect 5 to 10 map assets only

I need quotes or source cards

  • Open `quote-source-board.html`
  • Use the quote-card workflow
  • Start from existing card templates

I need AI to do first-pass work

  • Open `automation-hub.html`
  • Use one of the prefilled briefs if it already matches the task
  • Run one image, video, or text prompt template

Safe Session Checklist

Before You Start

  • Open the command center
  • Choose one lane only
  • Open the tracker if you will collect anything

During The Session

  • Work in a batch of 5 to 10 items
  • Save into the correct collection folder
  • Log source and license immediately

Before You Stop

  • Promote only the strongest items
  • Leave uncertain items in raw or ingest
  • Write nothing fancy, just keep it organized

Quick Entry Points

Collect Manuscripts

  • Read `example-workflow-manuscript-pull.md`
  • Start with rubricated pages or codex spreads
  • Judge them against codex and parchment motifs

Collect Maps

  • Read `example-workflow-jerusalem-map-pull.md`
  • Start with one Jerusalem batch
  • Keep geography clear and reusable

Build Quote Cards

  • Read `example-workflow-quote-card-build.md`
  • Start from the existing SVG templates
  • Keep hierarchy clear and archival

Use AI Tools Fast

  • Open `automation-hub.html`
  • Start with the Jerusalem or quote-card prefilled brief if applicable
  • Save prompt/run notes and review in `ai-review-dashboard.html` before approving outputs

Four Questions Before Keeping Anything

Fits The Look?

Would it sit naturally on one of the boards?

Reusable?

Can it support more than one episode or one recurring motif?

Traceable?

Can you log the source, URL, and license right now?

Stored Correctly?

Does it have a clear home in raw, curated, overlays, cutouts, or kits?
