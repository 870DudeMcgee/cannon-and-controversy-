# Run Logging Rules

Use this folder to keep an audit trail of AI-assisted work.

## What To Save Here

For each meaningful run, save:
- the date
- the tool used
- the prompt version
- the brief used
- which variants were accepted or rejected
- why the selected output survived

## Naming Suggestion

Use a folder or file name like:

`YYYY-MM-DD_task-tool-shortname_v001`

Example:

`2026-05-14_jerusalem-map-run_midjourney_v001.md`

## Minimum Log Format

- job name
- goal
- tool used
- prompt used
- references supplied
- result summary
- keep / reject decision
- destination folder if approved

## Why This Matters

Without logs, AI-assisted work vanishes into tool history and becomes hard to repeat.

With logs, you can:
- reuse what worked
- avoid rerunning bad prompts
- track which tools are actually saving time
- hand off the system to another person or another session without losing context
