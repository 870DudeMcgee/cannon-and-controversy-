# Quote Card / Source Caption Prompt Pack

Generate concise documentary-series text in a scholarly but cinematic tone.

Task goal:
Generate concise quote-card and source-caption text variants that fit the archival, scholarly tone of the documentary series and can drop into the existing reusable SVG templates.

Target use:
quote-card

Text inputs:
- primary text: Replace with the exact quotation or thesis line.
- secondary text: Replace with the speaker, source title, or source caption line.
- dates: Replace with date or period if needed
- names: Replace with speaker, author, institution, or location
- citations: Replace with exact citation text if needed

Style rules:
- concise, sober, and confident
- no clickbait language
- no exaggerated certainty
- readable on screen
- suitable for quote cards, chapter titles, lower thirds, or source captions

Generate:
- 5 variants
- one safest option
- one more dramatic but still credible option
- one shortest option

Must avoid:
- sensationalism
- modern marketing tone
- unsupported claims
- excessive length
