# Prompt Packs

These are copy-ready prompt files prefilled from the recurring briefs.

Use them when you do not want to translate a JSON brief into a prompt by hand.

## Fast Workflow

1. Open the matching brief if you need the structured context.
2. Open the matching prompt pack if you just want to paste into a tool fast.
3. Replace any clearly marked placeholder text.
4. Save the actual prompt used in a run log after generation.

## Rule

The prompt pack is the fast lane.
The brief remains the system of record.
