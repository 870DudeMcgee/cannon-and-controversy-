# Episode 05 Prompt Packs

Episode 05 focuses on where the removed books are referenced inside the Bible itself: direct quotations, strong allusions, and especially the Enochic patterns running through Jude, 2 Peter, Revelation, Hebrews, James, and the Gospels.

Use these packs to populate:
- opener and chapter-title variants
- quote cards and source strips for Jude, Enoch, Wisdom, Sirach, 2 Maccabees, and Assumption of Moses
- end-cards for the claim that the removed books remain embedded inside biblical discourse

This episode should feel like an internal-evidence reveal. The governing argument is that the removed books are not simply outside the Bible; they remain inside its language, logic, and quotations.
