# Episode 06 Prompt Packs

Episode 06 focuses on doctrinal consequences: resurrection, purgatory, the intermediate state, and the way canon decisions change what counts as legitimate afterlife language.

Use these packs to populate:
- opener and chapter-title variants
- doctrine comparison cards for the two-stage post-death model
- quote cards and evidence strips for Luke 16, 1 Enoch 22, 2 Maccabees 12, and Wisdom 3
- end-cards for the claim that canon choices shape doctrine outcomes

This episode should feel like a clarification episode, not a sectarian attack. The governing argument is that once books are removed, doctrine is forced to reorganize around a thinner textual base.
