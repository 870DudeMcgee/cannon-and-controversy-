# Lower Third / Source Strip Prompt Pack

Generate concise documentary-series text in a scholarly but cinematic tone.

Task goal:
Generate concise lower-third and source-strip text variants that can populate recurring documentary identifiers without breaking the series tone.

Target use:
other

Text inputs:
- primary text: Replace with subject or speaker name.
- secondary text: Replace with role, institution, location, or short descriptor.
- dates: none unless required
- names: replace with the exact person, institution, or location
- citations: optional

Style rules:
- concise, sober, and confident
- no clickbait language
- no exaggerated certainty
- readable on screen
- suitable for quote cards, chapter titles, lower thirds, or source captions

Generate:
- 5 variants
- one safest option
- one more dramatic but still credible option
- one shortest option

Must avoid:
- sensationalism
- modern marketing tone
- unsupported claims
- excessive length
