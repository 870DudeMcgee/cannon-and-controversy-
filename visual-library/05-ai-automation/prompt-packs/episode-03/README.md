# Episode 03 Prompt Packs

Episode 03 focuses on the Reformation cut: Luther, the Leipzig Debate, 2 Maccabees, sola fide, and the later Protestant hardening of the canon line.

Use these packs to populate:
- opener and chapter-title variants
- quote cards and source strips for Luther, Johann Eck, and James
- evidence cards for the canon criteria and the Reformation cut

This episode should feel like the moment the inherited canon tension becomes an explicit act of exclusion.
