# Episode 04 Prompt Packs

Episode 04 focuses on the deeper removed texts: Qumran, the Dead Sea Scrolls, 1 Enoch, and the wider textual ecosystem that later Western canons left outside the line.

Use these packs to populate:
- opener and chapter-title variants
- quote cards and evidence strips for Qumran, Cave 4, Cave 11, and 1 Enoch
- end-cards for the claim that exclusion did not mean irrelevance or non-circulation

This episode should feel like an archaeological reveal. The governing argument is that later exclusion cannot erase earlier authority, circulation, or survival value.
