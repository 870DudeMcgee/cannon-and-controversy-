# Fifth Pack Manifest

This pack adds transition plates and alpha-style mattes for codex openings and parchment reveals.

## Matte Assets

- `codex-page-matte-01.svg`
  - white page silhouette with black image window
  - use as a compositing mask for codex inserts
- `codex-page-matte-02.svg`
  - alternate page silhouette with a slightly tighter inner window
  - use for second-angle codex comps
- `parchment-reveal-matte-01.svg`
  - centered reveal matte from light center to dark edges
  - use for reveal timing tests and blend experiments

## Plate Assets

- `codex-opening-plate-01.svg`
  - ceremonial codex plate for source intros and chapter starts
- `parchment-reveal-plate-01.svg`
  - glowing parchment field for quote reveals and document intros
- `page-turn-slice-01.svg`
  - angled page slice for turn simulations and layered page-stack transitions

## Recommended Uses

1. Use `codex-page-matte-01.svg` with manuscript art inside `site-codex-frame.svg` derived comps.
2. Use `parchment-reveal-plate-01.svg` with `glow-veil-01.svg` and `dust-field-01.svg`.
3. Use `page-turn-slice-01.svg` as a top-layer element in codex-opening transitions.
