# Fourth Pack Manifest

This pack adds atmospheric overlays and transition surfaces that help the documentary feel cinematic without changing the core design language.

## Overlay Assets

- `dust-field-01.svg`
  - warm archival particle field
  - use over dark cards, maps, and chapter intros
- `glow-veil-01.svg`
  - restrained gold and ember bloom veil
  - use for parchment reveals and title emphasis
- `edge-burn-mask-01.svg`
  - scorched-edge vignette and frame darkening pass
  - use for transitions, historical rupture moments, and source reveals
- `manuscript-wash-01.svg`
  - red and green atmospheric wash derived from manuscript accent logic
  - use for subtle depth over parchment and codex comps

## Recommended Combinations

1. `parchment-background-01.svg` + `manuscript-wash-01.svg` + `illuminated-border-01.svg`
2. `parchment-background-02.svg` + `glow-veil-01.svg` + `title-card-red-gold-01.svg`
3. `source-citation-card-01.svg` + `dust-field-01.svg` + `gold-rule-02.svg`
4. map base + `edge-burn-mask-01.svg` + `timeline-spine-overlay-01.svg`

## Role In The System

These overlays are not stand-alone hero graphics. They are mood layers that make the recurring motifs feel tactile, aged, and ritualized.
