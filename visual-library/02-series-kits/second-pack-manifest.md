# Second Pack Manifest

This pack expands the reusable documentary language beyond backgrounds and borders into timeline, map, and citation systems.

## New Frame Elements

- `timeline-medallion-01.svg`
  - parchment-centered event badge with cross-axis geometry
- `timeline-medallion-02.svg`
  - ceremonial event badge with red-gold-green halo
- `map-marker-01.svg`
  - pointed destination marker for city or site emphasis
- `map-marker-02.svg`
  - medallion marker for calmer map labeling
- `gold-rule-01.svg`
  - minimal line separator
- `gold-rule-02.svg`
  - double-weight separator for titles and citations
- `gold-rule-03.svg`
  - ornament-centered separator

## New Episode-Neutral Templates

- `quote-card-template-01.svg`
  - large parchment quote card based on the callout and pull-quote logic from the site
- `source-citation-card-01.svg`
  - dark archival citation plate for textual witnesses and source intros

## Recommended Uses

1. Use `timeline-medallion-01.svg` and `gold-rule-03.svg` together for chronology sequences.
2. Use `map-marker-01.svg` on route maps where a destination needs dramatic emphasis.
3. Use `map-marker-02.svg` for calmer scholarly geography cards.
4. Use `quote-card-template-01.svg` for thesis lines and key quotations.
5. Use `source-citation-card-01.svg` whenever narration pivots into a cited witness.
