# First Pack Manifest

This is the first reusable visual kit derived from the site's existing language.

## Episode-Neutral Backgrounds

- `parchment-background-01.svg`
  - light parchment field
  - use for quote beds, source cards, and codex insert backgrounds
- `parchment-background-02.svg`
  - dark archival field with gold and ember bloom
  - use for chapter intros, dark title cards, and map backgrounds
- `title-card-red-gold-01.svg`
  - landscape master chapter card
  - use for episode sections and thesis statements
- `title-card-red-gold-02.svg`
  - vertical red/gold title card
  - use for reels, social clips, and mobile interstitials

## Frame Elements

- `illuminated-border-01.svg`
  - widescreen ceremonial border with corner ornament emphasis
- `illuminated-border-02.svg`
  - square border with stronger Ethiopian-inspired cross geometry
- `divider-rule-01.svg`
  - gold fade divider with centered glyph cluster
- `codex-silhouette-01.svg`
  - reusable codex page silhouette for chapter-open comps and source plates

## Recommended First Comps

1. Put `illuminated-border-01.svg` over `parchment-background-01.svg` for quote cards.
2. Put `title-card-red-gold-01.svg` over `parchment-background-02.svg` for chapter intros.
3. Put `codex-silhouette-01.svg` over `parchment-background-01.svg` and insert manuscript art inside the central window.
4. Use `divider-rule-01.svg` as a recurring separator across cards and lower thirds.
