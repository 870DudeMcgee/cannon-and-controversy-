What To Review Every Time

Style Match

  • Does it fit the boards immediately?
  • Does it preserve the archival tone?
  • Does it avoid generic fantasy or modern UI aesthetics?

Usability

  • Is it readable or legible fast enough?
  • Can it be reused beyond one moment?
  • Is the first-pass output worth refining?

Traceability

  • Was the prompt saved?
  • Was the tool used recorded?
  • Is the destination folder known before promotion?

Suggested Save Destinations

Generated References

  • Map studies that prove useful: `01-core-collections/maps/curated/` or `02-series-kits/motion-presets/`
  • Reusable quote-card systems: `02-series-kits/episode-neutral/`
  • Discarded or uncertain variants: keep logged in `05-ai-automation/runs/` instead of promoting them

Run Evidence

  • Save each meaningful run using the rules in `05-ai-automation/runs/README.md`
  • Keep prompt variations with the accepted result notes
  • Use one log file per meaningful run to keep handoff clean
