# Example Workflow: Quote Card Build

This is a build workflow rather than a sourcing workflow.

Goal: use the existing reusable kit to create one quote or source-card mockup without inventing a new design system every time.

## Step 1: Open The Right Surfaces

Open these first:

- `visual-library/index.html`
- `visual-library/quote-source-board.html`
- `visual-library/folder-map.html`
- `visual-library/04-docs/signature-motif-bible.md`

Why:
- the quote board shows the target look
- the folder map shows where reusable assets live
- the motif bible keeps the typography and border language consistent

## Step 2: Pick The Card Type

Choose one format only:
- quote card
- quote window card
- source citation card
- chapter or thesis plate

Do not mix formats in one first pass.

## Step 3: Start With Existing Reusable Assets

Use these existing kit files as your base:

- `02-series-kits/episode-neutral/quote-card-template-01.svg`
- `02-series-kits/episode-neutral/quote-window-card-01.svg`
- `02-series-kits/episode-neutral/source-citation-card-01.svg`
- `02-series-kits/frame-elements/source-tab-01.svg`
- `02-series-kits/frame-elements/source-tab-02.svg`
- `02-series-kits/frame-elements/illuminated-border-01.svg`
- `02-series-kits/frame-elements/divider-rule-01.svg`
- `02-series-kits/frame-elements/gold-rule-02.svg`

This means you are composing from a system, not designing from zero.

## Step 4: Decide The Information Hierarchy

For a quote card, keep only these text layers:
- the quotation itself
- the speaker or source
- an optional source line or date

For a citation card, keep only these:
- source title
- author or institution
- date or period
- optional short descriptor

Do not overload the card.

## Step 5: Follow The House Style

Use the recurring rules already defined in the library:
- parchment or charcoal bed, not bright white
- aged gold, not shiny metallic gold
- red as a signal accent, not a full background field
- strongest border detail at corners or openings
- readable text blocks with generous margins

## Step 6: Judge It Against The Quote Board

Ask:
- would this sit naturally next to the cards on `quote-source-board.html`?
- does it feel archival rather than modern social-media graphics?
- is the hierarchy clear in 2 seconds?

If no, simplify rather than decorate more.

## Step 7: Save The Result In The Right Place

If you are making a reusable base asset, it belongs in:
- `visual-library/02-series-kits/episode-neutral/`
- or `visual-library/02-series-kits/frame-elements/` if it is a modular component

If you are making a one-off test comp, keep it separate from the reusable library until it proves useful.

## Step 8: Connect It Back To The Motifs

The quote/source system is mainly tied to:
- illuminated borders
- glowing parchment reveal
- red/gold text reveals

If the card supports those motifs and stays episode-neutral, it is worth keeping.

## What Success Looks Like

A successful first quote-card session is not a finished episode graphic package.

It is:
- one clear card type chosen
- one existing template used
- one clean mockup composed
- the result still feeling like the rest of the library

That is enough.
