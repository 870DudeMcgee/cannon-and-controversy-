# Episode 06 starter pack

This pack supports `ep-06_resurrection-purgatory-and-the-intermediate-state`.

These files are a deliberately practical first pass built from source-backed manuscript material already gathered for Episode 08. They give you:
- direct 2 Maccabees support
- intertestamental manuscript atmosphere
- generic Greek manuscript texture for overlays
