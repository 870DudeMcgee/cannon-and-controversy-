# Episode 01 public-domain starter pack

This pack supports `ep-01_the-leveling-of-jerusalem`.

Included assets cover the strongest documentary needs for a first assembly pass:
- Josephus as the historian witness.
- Jerusalem city geography.
- Temple-area orientation.
- Destruction-of-Jerusalem aftermath imagery.
- Priestly / Caiaphas atmosphere support.

Still missing if you want a deeper pass later:
- stronger first-century archaeology plates
- Roman siege support plates
- cleaner Second Temple reconstruction imagery

Every file in this folder is listed in `manifest.csv` with source and license notes.
