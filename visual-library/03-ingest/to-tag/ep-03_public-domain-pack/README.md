# Episode 03 public-domain starter pack

This pack supports `ep-03_luther-and-the-reformation-cut`.

Included assets give you a usable first assembly layer for:
- Luther portrait beats.
- Luther as writer / translator.
- Johann Eck as opposing pole.
- Luther Bible artifact shots.

Still missing if you want a fuller pass later:
- stronger print-shop imagery
- Westminster / confessional document imagery
- Council of Trent visual support

Every file in this folder is listed in `manifest.csv` with source and license notes.
